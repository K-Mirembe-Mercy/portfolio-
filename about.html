My Story

The Journey
Behind the Code.

I started my coding journey with Python — curious, excited, and a little lost. But I kept going. Every error message was a lesson, every working program a victory.

My love for games drove me to build them: Balloon Shooters, Snake Games, Flappy Birds, Space Shooters, Egg Catchers — each one taught me something new about logic, loops, physics, and design.

I moved into web development — HTML, CSS, and JavaScript — and built real websites for real organizations like Codeon Africa and Mama Gracious. Seeing my work on a browser felt extraordinary.

Then came C programming: pointers, memory, structs. Challenging — but I built a complete Student Management System and more. Every hard thing I tackle makes me stronger.

Today I have 30+ projects, 3 GitHub badges, and an unstoppable drive to keep learning and building. I'm just getting started.
